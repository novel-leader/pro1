파이선 정리 - 자료형

  • str

    • 문자열은 기본적으로 '', ""로 감싼다.
    • 백슬래시(\)로 ', "를 문자열 안에 포함할 수 있다. (\', \")
      ※ 이스케이프 코드 : \n(줄바꿈), \t(탭간격), \(\)그대로 포함
    • ''' ''', """ """를 사용하면 이스케이프 코드 불필요
    • 순서가 있기 때문에 인덱스(a[6]), 슬라이싱(a[2:8]) 가능(sequence)
    • 변경이 불가능(immutable)해서 인덱스로 한 글자 수정 불가
    • 문자열은 더하기(+), 곱하기(*) 가능
    • f문자열 포매팅 가능. f“나는 {A}입니다.”

  • list

    • L = [1, 2, 3, 4, 5]
    • 여러 자료형을 담을 수 있다.
    • 순서가 있어서 인덱스, 슬라이싱 가능
    • 다중리스트 인덱스(a[3][2][4])
    • 변경 가능해 삽입(append), 삭제(remove), 정렬(sort) 가능 (mutable)
    • 더하기(+), 곱하기(*) 가능

  • tuple

    • t = (1, 2, 3, 4, 5)
    • 리스트와 거의 동일하나 변경이 불가능
    • 변경이 불가능하기 때문에 사용할 수 있는 메소드 수도 적음
    • 속도 빨라서 파이썬 내부적으로 자주 사용
    • 스왑(swap)을 쉽게 할 수 있다. (a, b = b, a)

  • dict

    • d = {‘name’:‘abc’, ‘birth’:2000}
    • set의 일종으로 키:값 쌍으로 구성된다.
    • 키는 중복될 수 없다. (값은 중복 가능)
    • 인덱스가 없고 키가 인덱스 역할을 함
    • for 반복문을 이용해 사전(딕셔너리) 내용(키와 값)을 출력할 수 있다.

  • set

    • s = {1, 2, 3, 4, 5}
    • 수학에서 말하는 집합과 유사하다.
    • 순서가 없어서 익덱스, 슬라이싱 불가
    • 값 중복 불가, 변경 가능
    • in, not in 을 사용하여 집합 포함여부를 알 수 있다.
