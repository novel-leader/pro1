SSH공개키 사용방법

SSH공개키 생성 및 등록

ssh공개키 있는지 확인
$ cd ~/.ssh :폴더이동
$ ls :폴더내 파일 확인

공개키 생성
$ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

첫 질문 : 공개키 저장위치
두번째 질문 : 비밀번호 설정(그냥 엔터 가능)

$ ls : 폴더내 파일 확인
$ clip < ~/.ssh/id_ed25519.pub :파일 복사

github 등록

연결테스트
$ ssh -T git@github.com

에러발생시 원인 및 해결방법

port 22: Connection timed out

보안이 강한 회사는 포트가 막혀있어서 ssh로 연결불가
에러내용 :
ssh: connect to host github.com port 22: Connection timed out
여기에서 포트22가 외부연결을 막아놓은것임
